# API nhonga.net - Funcionalidades Principais

# 1. Sistema de Autenticação e Utilizadores
- Registo e login de utilizadores 

- Gestão de perfis profissionais (tipo LinkedIn)

- Upload de fotos de perfil e currículos

- Sistema de competências e recomendações

# 2. Módulo de Empregos
- CRUD de vagas de emprego

- Sistema de candidaturas

- Filtros por categoria, localização, tipo de contrato

- Alertas de novas vagas

- Gestão de empresas recrutadoras

# 3. Módulo Académico
- Gestão de bolsas de estudo

- Cursos online e presenciais

- Estágios e oportunidades de pesquisa

- Candidaturas universitárias

- Feiras de ciência e palestras

# 4. Marketplace de Serviços
- Catálogo de prestadores de serviços

- Sistema de orçamentos e propostas

- valiações e comentários

- Portfólios de profissionais

- Categorias (limpeza, reparações, educação, etc.)

# 5. Rede Social Profissional (Networking)
- Sistema de posts e feed

- Likes, comentários e partilhas

- Conexões entre profissionais

- Convites e sugestões de contactos

- Grupos de discussão por área

# 6. Sistema de Pesquisa e Filtros
- Pesquisa global por palavras-chave

- Filtros avançados por categoria, localização, tipo

- Sugestões de pesquisa

- Contadores de resultados

# 7. Notificações e Comunicação
- Sistema de mensagens entre utilizadores

- Notificações de candidaturas, convites, etc.

- Alertas personalizados

#8. Gestão de Conteúdo
- Upload de imagens e documentos

- Moderação de conteúdo

- Categorização automática

A API deve suporta todas estas funcionalidades com endpoints RESTful, 
autenticação JWT, validação de dados, e integração com base de dados 
para armazenar utilizadores, vagas, serviços, posts, e
todas as interações da plataforma.
